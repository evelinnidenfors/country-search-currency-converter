# Country Search App
## Introduction
Allows you to search for countries. The results page displays:
* Flag
* Country name
* Capital
* Population
* Currency

You are able to convert the local currency to SEK by entering an amount in the input field.

## Technologies
* React
* JSX
* ES6
* CSS

## Launch
To run the project follow the below steps:
```
$ npm install
$ npm start
```
